# "점" 세계관

## 컨셉

서비스명 "점하나"가 곧 메인 캐릭터 이름. 서비스 확장 시 캐릭터가 추가되는 세계관 구조.

---

## 점 씨네 가족 — 점술사 가문

| 캐릭터 | 이름 | 성/이름 | 역할 | 시점 |
|---|---|---|---|---|
| 맏이 | 점하나 | 점 / 하나 | 타로 리더 | MVP |
| 둘째 | 점둘이 | 점 / 둘이 | 사주 분석가 | Phase 2 |
| 셋째 | 점삼이 | 점 / 삼이 | 미정 (별자리/꿈해몽 등) | Phase 3+ |

> 캐릭터가 늘어날수록 서비스가 확장되는 구조. "점하나의 동생 점둘이를 소개합니다" 같은 자연스러운 마케팅 가능.

---

## 점하나 캐릭터 (MVP)

### 성격
- 친근하고 장난기 있는, "해볼까?" 톤
- 가볍고 편한 친구 같은 존재
- 서비스 내 안내 문구 = 점하나의 말투

### 형태
- 동그란 점 + 눈 두 개 + 타로 리더 정체성을 나타내는 소품 (작은 마녀 모자, 별 장식 등)
- 밝고 귀여운 일러스트 스타일 (카드 이미지와 톤 통일)
- 표정 바리에이션으로 상황별 감정 표현

### 제작
- 카드 이미지와 동일하게 AI 생성 (Midjourney/DALL-E)
- 표정별 바리에이션 제작 (아래 등장 위치 참조)

---

## 서비스 내 등장

| 위치 | 점하나의 모습 | 말투/문구 |
|---|---|---|
| 로고 | "점하나" 텍스트 + 캐릭터 | - |
| 파비콘/앱 아이콘 | 점하나 얼굴 | - |
| 공통 로딩 화면 | 통통 튀는 애니메이션 | (문구 없음) |
| 카드 뽑기 진입 | 두근두근하는 모습 | "어떤 카드가 나올까?" |
| 결과 스트리밍 중 | 해석 중인 모습 | - |
| 히스토리 빈 상태 | 손 흔드는 모습 | "여기는 아직 비어있어요. 점 하나 찍으러 가볼까요?" |
| 오류 모달 | 미안한 표정 | "미안, 잠시 후 다시 해볼까?" |
| 공유 이미지 | 워터마크 | - |

## 확장 시나리오

### 사주 서비스 출시 (Phase 2)
- "점하나의 동생 점둘이를 소개합니다"
- 앱 홈에서 캐릭터 선택 → 해당 서비스 진입
- 점둘이는 점하나와 다른 성격/말투 (차분한, 진지한 등)

### 추가 서비스 (Phase 3+)
- 점삼이 등 새 캐릭터 추가
- 캐릭터별 개성 차별화
- 시즌별 의상/소품 변경 가능 (이벤트)
