# 벤치마킹 분석: 점신 타로 (jeomsin.co.kr/tarothome)

> 분석일: 2026-05-01

---

## 1. 서비스 구조 요약

점신은 운세/타로/사주를 통합 제공하는 국내 1위 운세 앱. 타로 섹션은 **40개 이상의 테마별 타로 콘텐츠**를 제공하며, 광고 기반 무료 모델로 운영.

### 하단 탭 네비게이션 (5개)
| 탭 | 설명 |
|---|---|
| 점신 | 메인 홈 (운세 종합) |
| 2026년 운세 | 연간 운세 |
| 타로 | 타로 전용 홈 |
| 상담 | 전문 상담사 연결 (유료) |
| 점신몰 | 굿즈/부적 쇼핑 |

## 2. 타로 홈 구성

### 섹션 1: "매일 찾게 되는 타로카드" (일상 타로)
- **오늘의 타로** — "오늘 하루는 어떨까?"
- **이달의 타로** — "이번달 나의 총운은?"
- "타로 점 보기 >" CTA 버튼

### 섹션 2: "타로점으로 고민해결!" (테마별 타로)
- 탭 필터로 카테고리 분류
- 각 타로 콘텐츠에 **해시태그 2~3개** 표시 (예: #시작 #변화 #확장)
- 썸네일 이미지 + 제목 카드 형태

### 카테고리 탭 (6개)

| 카테고리 | 대표 콘텐츠 |
|---|---|
| 전체 | 모든 타로 콘텐츠 |
| 26년 타로 | 운/사랑/재물/취업/학업/건강/비즈니스 흐름 (7개) |
| 연애/결혼 | 이상형, 연애스타일, 매력, 연애전략, 주의점 등 (10+개) |
| 짝사랑/재회 | 상대방 마음, SNS 염탐, 재회운, 결혼 가능성 등 (8+개) |
| 취업/학업 | 적성, 능력, 주변 도움, 취업 성공 여부 등 (5+개) |
| 그 외 | 귀인, 뒷얘기, 이동수, 이미지, 전성기, 반려동물, 마음건강 등 (10+개) |

### 총 콘텐츠 수: 40개 이상

## 3. 사용자 흐름 (UX Flow)

```
[1. 타로홈] → [2. 테마 선택] → [3. 카드 3장 선택] → [4. 광고 시청] → [5. 결과 보기]
```

### 세부 흐름

#### 1단계: 타로홈
- 일상 타로 (오늘/이달) + 테마별 타로 목록
- 카테고리 탭 필터로 빠른 탐색
- 각 콘텐츠에 해시태그로 미리보기 제공 → 호기심 유발

#### 2단계: 테마 선택
- 탭하면 바로 카드 선택 화면으로 진입 (질문 선택 단계 없음)
- 테마 자체가 이미 질문을 내포 (예: "그 사람은 왜 갑자기 멀어졌을까?")

#### 3단계: 카드 선택
- 상단 안내: "오늘의 기운을 생각하며 카드 3장을 선택해주세요."
- 상단에 결과 슬롯 표시: 애정운 / 재물운 / 학업&취업운
- 카드 22장 뒷면 나열 (메이저 아르카나만 사용)
- 터치하면 선택됨 (뒤집기 애니메이션은 결과에서)
- 3장 선택 후: "다시 선택" / "결과 보기" 버튼

#### 4단계: 광고 시청 (결과 게이트)
- "광고 시청하고 결과보기" 텍스트
- 배너 광고 2개 표시 (Google AdSense)
- 광고 시청 완료 후 결과 화면 전환

#### 5단계: 결과 (미확인 — 광고 시청 필요)
- 카드 이미지 + 해석 문장 (추정)

### 특이 사항
- **"카드설정" 버튼** — 카드 덱 디자인 변경 가능 (커스터마이징)
- 질문 입력 단계 없음 — 테마가 곧 질문
- 스프레드 선택 없음 — 모든 테마가 3장 고정
- 결과를 보려면 반드시 광고 시청

## 4. 수익 모델 분석

| 수익원 | 설명 | 단계 |
|---|---|---|
| 광고 (메인) | 결과 보기 전 필수 광고 시청 + 배너 | 핵심 |
| 전문 상담 | 24시 상담사 연결, 가입시 무료 쿠폰 | 유료 전환 |
| 행운패스 | 광고 제거 구독 모델 | 구독 |
| 점신몰 | 부적/굿즈 판매 | 커머스 |

### 핵심 인사이트
- **"무료 체험 → 광고 게이트 → 결과"** 패턴이 핵심
- 타로 자체는 완전 무료 (기회 제한 없음), 대신 매번 광고 시청
- 전문 상담으로의 유료 전환 유도 (타로 → "더 깊은 상담 필요하면?")

## 5. 우리 서비스와 비교 & 시사점

### 점신이 잘하는 것 (벤치마킹 포인트)

| 점신 | 우리 현재 설계 | 개선 방향 |
|---|---|---|
| 40+ 테마별 타로 (질문이 곧 콘텐츠) | 11개 사전 질문 | 테마 기반 콘텐츠로 확장 |
| 스프레드 선택 없음 (3장 고정) | 스프레드 선택 단계 있음 | MVP에서 3장 고정으로 단순화 검토 |
| 카테고리 탭 필터 | 카테고리별 질문 목록 | 탭 필터 UI 도입 |
| 해시태그로 콘텐츠 미리보기 | 없음 | 해시태그/키워드 추가 |
| 카드덱 커스터마이징 | 없음 | Phase 2 고려 |
| 매번 광고 (기회 제한 없음) | 1회 무료 → 기회 제한 | 수익 모델 재검토 |
| 테마가 곧 질문 (1단계 생략) | 질문 선택 단계 별도 | 테마 탭 → 바로 카드 선택 |

### 우리가 차별화할 수 있는 것

| 차별점 | 설명 |
|---|---|
| AI 개인화 해석 | 점신은 정적 해석, 우리는 AI 생성 해석으로 매번 다른 결과 |
| 정/역방향 | 점신은 정방향만 (확인 필요), 우리는 정/역방향으로 해석 깊이 2배 |
| 스트리밍 해석 | AI가 실시간으로 해석을 써내려가는 경험 (몰입감) |
| 결과 공유 | 카드 이미지 + 핵심 해석 요약을 이미지로 생성 → 바이럴 |
| 마이너 아르카나 | 점신은 메이저 22장만, 우리는 78장 전체 활용 가능 |

## 6. MVP 설계 반영 제안

### 즉시 반영 (MVP 변경)

#### 6-1. 사용자 흐름 단순화
```
AS-IS: [홈] → [스프레드 선택] → [질문 선택] → [카드 뽑기] → [결과]
TO-BE: [홈 (테마 목록)] → [카드 뽑기] → [광고] → [AI 해석 결과]
```
- 스프레드 선택 제거 → MVP는 3장(쓰리카드) 고정
- 질문 선택 제거 → 테마가 곧 질문 (점신 방식)
- 단계 축소: 5단계 → 3단계

#### 6-2. 테마 기반 콘텐츠 구조
기존 "사전 질문 목록"을 "테마 콘텐츠"로 전환:

| 카테고리 | MVP 테마 (최소) |
|---|---|
| 일상 | 오늘의 타로, 이번 주 타로 |
| 연애 | 나의 연애운, 상대방의 마음, 새로운 인연 |
| 직장 | 직장 전망, 이직 타이밍 |
| 재물 | 나의 재물운 |
| 학업 | 시험/학업 결과 |
| 일반 | 지금 필요한 메시지, 이 선택이 맞을까 |

→ 총 11개 테마 (기존 질문 수 유지하되 구조만 변경)

#### 6-3. 수익 모델 조정
```
AS-IS: 1회 무료 → 기회 소진 → 광고/결제로 기회 추가
TO-BE: 매회 무료 → 결과 보기 전 광고 시청 필수 (점신 방식)
```
- 기회 제한 모델 제거 → 무제한 이용 + 매번 광고
- 장점: 사용 허들 낮춤, 반복 사용 유도, 광고 노출 극대화
- UserChance 데이터 구조 불필요 (MVP 단순화)

#### 6-4. 홈 화면 구조 변경
```
AS-IS: 서비스 소개 + "타로 보기" CTA
TO-BE: 일상 타로 섹션 + 카테고리 탭 + 테마 카드 목록
```
- 진입 즉시 콘텐츠 목록 노출 → 탐색 유도
- 해시태그로 테마 미리보기

### Phase 2 이후

- 카드덱 커스터마이징 (덱 디자인 변경)
- 연간/월간 타로 (시즌 콘텐츠)
- 전문 상담 연결 (수익 다각화)
- 원카드 스프레드 추가 (빠른 체험용)

## 7. 카드 뽑기 UI 참고

### 점신 방식
- 22장 카드 뒷면을 4열 x 5행 + 2장 그리드로 배치
- 다크 톤 배경 + 신비로운 무드
- 상단에 결과 슬롯 3개 (애정운/재물운/학업&취업운) 미리 표시
- 선택한 카드에 체크 표시
- 하단: "카드를 선택해 주세요" → 3장 선택 후 "다시 선택" / "결과 보기"

### 우리 서비스 적용
- 카드 배치: 그리드 레이아웃 (모바일 4열, 데스크탑 6열)
- 선택 시 뒤집기 애니메이션 추가 (점신보다 인터랙션 강화)
- 상단 슬롯: 과거/현재/미래 (또는 테마에 따라 다르게)
- 배경: 다크 + 별/우주 테마로 몰입감 제공
