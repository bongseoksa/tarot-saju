## 공통 컴포넌트 디자인 패턴

### 테마 카드 (ThemeCard)

시안 기준 구조 (`home/code.html`):
```
┌─┬──────────────────────────┬───┐
│▌│ 제목 (bold)               │ > │
│▌│ 설명 (text-secondary)     │   │
│▌│ [#태그1] [#태그2]         │   │
└─┴──────────────────────────┴───┘
 ↑ 카테고리별 컬러바 (2px)
```

카테고리별 컬러바:
- 일상: `#6B38D4` (primary)
- 연애: `#F8C8D8` (pastel-pink)
- 직장: `#2B6CB0` (info)
- 재물: `#D4920B` (warning)
- 학업: `#2D8A4E` (success)

### 광고 슬롯 (Ad Slot)

```
┌─────────────────────────────────┐
│                        [광고]   │  ← 우상단 라벨
│          광고 영역               │  ← #F0F0F0 bg
│                                 │  ← border-dashed
└─────────────────────────────────┘
```
- 콘텐츠 카드와 시각적으로 구분 (bg색, border 스타일)
- "광고" 라벨 필수 (text-[10px], 우상단 또는 좌상단)
