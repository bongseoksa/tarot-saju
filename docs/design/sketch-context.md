# 점하나 — 디자인 시안 컨텍스트

> 이 문서는 Sketch AI에 전달하기 위한 디자인 컨텍스트 요약본입니다.
> 모든 내용은 확정된 기획/정의 기반으로 작성되었습니다.

---

## 1. 서비스 개요

- **서비스명**: 점하나 (JeomHana)
- **한줄 소개**: AI가 읽어주는 타로 카드. 무료, 무제한.
- **핵심 가치**: 가볍게 한번 해보는 타로점. 무겁지 않은 신비로움.
- **타겟 유저**: 20~30대, 타로/운세에 가벼운 흥미가 있는 일반 사용자
- **플랫폼**: 모바일 웹 우선 (SPA), 이후 Android/iOS 앱 (WebView 래핑)

## 2. 브랜드 아이덴티티

### 브랜드 의미
> "오늘, 점 하나 찍어볼까?"

- 일상의 가벼운 한마디 — 친구에게 말하듯 가볍게 시작하는 점술
- 시작점(點) — 카드 한 장의 터치가 오늘의 방향을 만든다
- 점(占) + 하나(一) — 가장 먼저 떠오르는 점술 서비스

### 브랜드 톤
- 말투: 친구처럼 편한, "~해볼까?"
- 분위기: 무겁지 않은 신비로움
- 포지셔닝: 심각한 점술 X → 가볍게 한번 해보는 점술 O

### 서비스 내 카피
- 앱 소개: "점 하나 찍으면, 오늘이 달라질지도"
- 공유 메시지: "점하나에서 타로 봤는데 소름..."
- 로딩 화면: "당신을 위한 점, 하나 준비 중"
- 결과 화면 CTA: "점 하나 더 찍어볼까?"
- 히스토리 빈 상태: "여기는 아직 비어있어요. 점 하나 찍으러 가볼까요?"
- 오류 모달: "일시적으로 해석을 불러오지 못했어요. 잠시 후 다시 시도해주세요."

## 3. 캐릭터 — 점하나

서비스의 마스코트이자 안내자. 서비스명 = 캐릭터 이름.

### 형태
- 동그란 점 + 눈 두 개 + 작은 마녀 모자 + 별 장식
- 밝고 귀여운 일러스트 스타일 (카드 이미지와 톤 통일)
- 표정 바리에이션으로 상황별 감정 표현

### 등장 위치별 모습

| 위치 | 점하나의 모습 | 문구 |
|---|---|---|
| 로고 | "점하나" 텍스트 + 캐릭터 | — |
| 파비콘/앱 아이콘 | 점하나 얼굴 (단순화) | — |
| 공통 로딩 화면 | 통통 튀는 애니메이션 | (문구 없음) |
| 카드 뽑기 진입 | 두근두근하는 모습 | "어떤 카드가 나올까?" |
| 결과 스트리밍 중 | 해석 중인 모습 | — |
| 히스토리 빈 상태 | 손 흔드는 모습 | "여기는 아직 비어있어요..." |
| 오류 모달 | 미안한 표정 | "미안, 잠시 후 다시 해볼까?" |
| 공유 이미지 | 워터마크 | — |

### 제작 방식
- AI 생성 (Midjourney/DALL-E)
- 앱 아이콘: 1024x1024 원본, 파비콘 ico(16/32) + apple-touch-icon(180) + webmanifest(192/512)

## 4. 디자인 톤 & 무드

### 무드 키워드
밝고 귀여운 / 가볍게 접근 / 카테고리별 분위기 전환

| 요소 | 방향 |
|---|---|
| 전체 톤 | 밝은 베이스 (라이트 모드) — 친근하고 부담 없는 진입 |
| 무게감 | 가벼움, 귀여움 — 점술 앱이 아닌 감성 앱 느낌 |
| 홈 화면 | 밝은 배경 + 컬러풀한 테마 카드 |
| 결과 화면 | 밝은 배경 유지 — 해석 텍스트 가독성 최적 |
| 카드 선택 화면 | 카테고리별 분위기 전환 (파스텔 배경 색감 변화) |

### 레퍼런스
- 토스/당근 같은 밝고 친근한 앱 톤 + 카테고리별 감성 포인트
- 별자리 앱(Co-Star, The Pattern)의 미니멀 감성 참고

### 피해야 할 것
- 금색 테두리, 한자, 부적 이미지 등 전통 점술 클리셰
- 과도한 이펙트 (반짝이, 별 파티클 등)
- 전체 다크 모드 (가벼운 진입 톤과 불일치)

| 직장 | 소프트 블루 `#DBEAFE` | 미디엄 블루 `#3B82F6` |

## 7. 페이지별 화면 설계

### 네비게이션 원칙
- 탭바 없음 — 선형 플로우, 헤더 + 뒤로가기 방식
- 설정 페이지 없음

### 사용자 흐름
```
[1. 홈] → [2. 카드 뽑기] → [3. 광고(오버레이)] → [4. 결과] → [5. 공유/저장]
```

### 7-1. 홈 (`/`)

**구조:**

| 영역 | 배치 |
|---|---|
| 헤더 | 좌: 점하나 로고(캐릭터+텍스트), 우: 히스토리 아이콘 |
| 퀵 진입 | 홈 상단 — "오늘의 타로 한 장 뽑아볼까?" + CTA 버튼 (인기 테마 1개 바로 진입) |
| 카테고리 탭 | 가로 스크롤 필터 (전체/일상/연애/직장/재물/학업/기타) |
| 테마 카드 리스트 | 세로 스크롤, 카드형 (테마 제목 + 한줄 설명 + 카테고리 태그) |
| 푸터 | 개인정보처리방침 · 이용약관 링크 |

**테마 콘텐츠 (11개):**

| 카테고리 | 테마 | 해시태그 |
|---|---|---|
| 일상 | 오늘의 타로 | #오늘 #하루운세 |
| 일상 | 이번 주 타로 | #이번주 #주간운세 |
| 연애 | 나의 연애운 | #연애 #애정운 |
| 연애 | 그 사람의 마음 | #짝사랑 #속마음 |
| 연애 | 새로운 인연 | #인연 #만남 |
| 직장 | 직장에서의 전망 | #직장 #커리어 |
| 직장 | 이직 타이밍 | #이직 #전환 |
| 재물 | 나의 재물운 | #재물 #금전운 |
| 학업 | 시험/학업 결과 | #시험 #합격 |
| 기타 | 지금 필요한 메시지 | #조언 #메시지 |
| 기타 | 이 선택이 맞을까 | #선택 #결정 |

**상태:**
- 기본: 전체 테마 표시
- 카테고리 필터 활성: 선택된 카테고리의 테마만 표시
- 미완료 세션 모달: "확인하지 못한 결과가 있어요" + [이어서 보기] + [닫기] + □ 더 이상 보지 않기

### 7-2. 카드 뽑기 (`/reading/:themeId`)

**구조:**

| 영역 | 배치 |
|---|---|
| 헤더 | 좌: 뒤로가기, 중앙: 테마 제목 |
| 상단 슬롯 | 가로 3칸 — 과거/현재/미래 라벨, 선택된 카드가 슬롯으로 이동 |
| 카드 그리드 | 22장, 4열 배치 (4x5 + 하단 2장 중앙 정렬), **그리드 영역만 세로 스크롤** |
| 하단 버튼 | 3장 선택 완료 시 활성화 — "점 보기", **하단 고정** |

**레이아웃 원칙:**
- 상단 슬롯(3칸) + 하단 버튼은 화면에 고정 (sticky)
- 카드 그리드 영역만 세로 스크롤 — 소형 기기에서도 카드 크기를 충분히 유지
- 카드 크기: 터치 타겟 최소 44x44 확보, 2:3 비율 유지
- Safe Area: iOS 노치 + 홈 인디케이터, 모바일 브라우저 주소창/툴바 고려 (`safe-area-inset` 적용)

**배경:** 카테고리별 파스텔 색감으로 전환 (5번 컬러 팔레트 참조)

**카드 선택 경험 (4단계 애니메이션):**

1. **기대감** — 카드에 손이 가까워지면(호버/터치) 카드가 살짝 떠오름. 미세한 빛남 효과
2. **몰입** — 터치 시 카드가 위로 떠오르며 뒤집힘. 주변 카드가 살짝 어두워짐 (선택 카드에 집중). 짧은 스케일 펄스(1.05x → 1.0x)
3. **설렘** — 카드 앞면 공개 + 부드러운 글로우 + 카드명 아래에서 페이드인
   - 정방향: 카테고리 포인트 색상 글로우
   - 역방향: 카드가 거꾸로(180도) 공개 + 보라 글로우 + "역방향" 라벨 페이드인
4. **완성감** — 카드가 상단 슬롯으로 부드럽게 이동 (스프링 커브) + 가벼운 바운스 + 라벨 활성화

**상태:**
- 선택 전 (0/3): 모든 카드 뒷면, 슬롯 비어있음
- 선택 중 (1/3, 2/3): 선택된 카드는 슬롯에, 나머지 그리드에서 반투명 처리
- 선택 완료 (3/3): 하단 "점 보기" 버튼 활성화
- 뒤집기 중: 추가 탭 무시

### 7-3. 광고 게이트 (오버레이)

- 별도 페이지 아님, 전면 오버레이
- 공통 로딩 화면 (터치/뒤로가기 차단) → 전면 광고(Interstitial) 재생 → 완료 시 결과 이동
- 실패 시 조용히 스킵 → 바로 결과 이동

### 7-4. 결과 (`/result/:resultId`)

**구조:**

| 영역 | 배치 |
|---|---|
| 헤더 | 좌: 뒤로가기(홈), 중앙: 테마 제목 |
| 카드 요약 | 상단 — 가로 3장 나열 (과거/현재/미래 라벨 + 카드 이미지 + 카드명 + 정/역방향) |
| 해석 텍스트 | 카드 요약 아래 — AI 스트리밍 타이핑 영역, 세로 스크롤 |
| 배너 광고 | 하단 영역 (결과 페이지에만 배치, 홈에는 없음) |
| 하단 고정 | 공유 버튼 + "점 하나 더 찍어볼까?" CTA |

**상태:**
- AI 스트리밍 중: 타이핑 효과로 텍스트가 한 글자씩 나타남 (점하나 캐릭터 "해석 중" 모습)
- 완료: 공유 버튼 활성화, 결과 자동 저장
- 실패: 오류 모달 (점하나 미안한 표정) → 홈 이동

**AI 해석 구조:**
- 과거 카드 해석
- 현재 카드 해석
- 미래 카드 해석
- 종합 조언
- 한줄 요약

---

### 7-5. 히스토리 (`/history`)

**구조:**
- 헤더: 좌 뒤로가기 + "히스토리"
- 결과 목록: 최신순, 최대 20건, 각 항목은 테마명 + 날짜 + 카드 3장 썸네일
- 항목 탭 → 결과 페이지 이동
- 빈 상태: 점하나 캐릭터 (손 흔드는 모습) + "여기는 아직 비어있어요. 점 하나 찍으러 가볼까요?" + 홈 이동 버튼

---

### 7-6. 공유 결과 (`/shared/:shareId`)

- 결과 페이지와 동일한 레이아웃 (카드 요약 + 해석 전문)
- 하단 CTA: "나도 점 하나 찍어볼까?" → 홈 이동
- 공유/저장 버튼 없음 (공유받은 사람은 조회만)

---

### 7-7. 공통 요소

#### 공통 로딩 화면
- 점하나 캐릭터 통통 튀는 애니메이션
- 문구 없음, 터치/뒤로가기 차단
- 광고 로드, AI 호출 등 비동기 대기에 공통 사용

#### 오류 모달
- 점하나 미안한 표정
- "일시적으로 해석을 불러오지 못했어요. 잠시 후 다시 시도해주세요."
- [홈으로 돌아가기] 버튼

#### 미완료 세션 모달 (홈 진입 시)
- "확인하지 못한 결과가 있어요"
- 미완료 세션 목록 (1~3건, 테마명 표시)
- [이어서 보기] 버튼
- [닫기] 버튼
- □ 더 이상 보지 않기 체크박스

## 8. 카드 이미지 에셋

### 필요 에셋
- 메이저 아르카나 22장 (앞면) + 카드 뒷면 1장 = 총 23장

### 규격
- 비율: 2:3
- 포맷: WebP
- 구체적 픽셀 사이즈는 스케치에서 확정

### 참고 덱
- **라이더-웨이트(Rider-Waite)** 타로 — 가장 널리 쓰이는 표준 덱. 각 카드의 전통적 상징과 구도를 기반으로 하되, 점하나 스타일로 재해석
- 원본 해설: *The Pictorial Key to the Tarot* (A.E. Waite, 1911)

---

### 카드 앞면 — 공통 디자인 원칙

**핵심: 점하나 마스코트가 모든 카드의 주인공**

- 22장 모두 점하나 캐릭터(동그란 점 + 눈 + 마녀 모자)가 해당 카드의 역할을 연기하는 구도
- 라이더-웨이트의 전통적 상징 요소를 귀엽게 재해석하여 배경/소품에 배치
- 점하나의 표정과 포즈로 카드의 감정/에너지를 전달

**스타일 가이드:**
- 스타일: flat illustration / cute / pastel / minimal
- 톤: 밝고 따뜻한, 전통 타로 클리셰(다크/금색/한자) 완전 배제
- 배경: 각 카드의 상징에 맞는 파스텔 배경 (단색 또는 그라데이션)
- 선: 부드러운 라운드 라인, 굵은 아웃라인 없음
- 질감: 플랫, 그림자 최소화
- 22장 일관성 유지를 위해 동일 프롬프트 베이스 + 카드별 변수만 변경
- 제작: AI 생성 (Midjourney/DALL-E)

**카드 레이아웃:**

```
┌─────────────────┐
│                  │
│   (상징 배경)     │
│                  │
│    [점하나 캐릭터  │
│     + 소품/포즈]  │
│                  │
│                  │
├─────────────────┤
│  카드명 (한글)    │
│  로마 숫자       │
└─────────────────┘
```

- 상단 ~70%: 일러스트 영역 (점하나 + 상징 요소)
- 하단 ~30%: 카드명(한글) + 로마 숫자, 브랜드 Primary 컬러 배경

---

### 카드 뒷면 — 공통 디자인

- 모든 카드 공통 (카테고리 무관)
- 중앙: 점하나 심볼 (동그란 점 + 마녀 모자)
- 배경: Primary 컬러(소프트 바이올렛/라벤더) 기반 패턴
- 패턴: 별/달/점 등 점술 감성의 미니멀 패턴 반복
- 테두리: 부드러운 라운드 코너
- 뒷면만 보고도 "점하나 카드"임을 알 수 있는 브랜딩

---

### 카드별 디자인 가이드 (22장)

> 각 카드에서 점하나 캐릭터가 어떤 모습/포즈를 하고, 어떤 상징 소품을 배치할지 정의.
> 라이더-웨이트 전통 상징을 기반으로 하되, 귀엽고 밝은 톤으로 재해석.

#### 0. 광대 (The Fool) — 여행자 점하나
- **점하나 모습**: 작은 보따리를 들고 한 발을 내딛으려는 모습. 호기심 가득한 표정
- **배경/소품**: 파스텔 하늘, 절벽 가장자리, 흰 꽃(장미 대신), 작은 강아지가 발치에서 뛰어다님
- **색감**: 파스텔 옐로우 + 스카이블루
- **핵심 상징**: 절벽, 흰 꽃, 작은 개, 보따리, 먼 산
- **점성**: 천왕성 / 공기 원소

#### 1. 마법사 (The Magician) — 마법사 점하나
- **점하나 모습**: 한 손을 하늘로, 다른 손을 아래로. 집중하는 진지한 표정
- **배경/소품**: 작은 테이블 위에 완드/컵/소드/펜타클 미니 아이콘. 머리 위 무한대(∞) 기호 반짝임
- **색감**: 파스텔 레드 + 골드(밝은 톤)
- **핵심 상징**: 무한대 기호, 네 원소 도구, 붉은 장미와 백합
- **점성**: 수성

#### 2. 여사제 (The High Priestess) — 신비로운 점하나
- **점하나 모습**: 두루마리를 무릎에 올리고 앉아 있는 모습. 차분하고 깊은 눈빛
- **배경/소품**: 두 기둥(B/J 레터링), 초승달 왕관, 뒤로 석류/별 패턴 장막
- **색감**: 파스텔 네이비 + 실버 (밝게)
- **핵심 상징**: 두 기둥, 토라 두루마리, 초승달 왕관, 석류 장막
- **점성**: 달

#### 3. 여황제 (The Empress) — 풍요로운 점하나
- **점하나 모습**: 꽃밭에서 편안하게 앉아 있는 모습. 행복한 미소
- **배경/소품**: 밀밭, 흐르는 시냇물, 별 12개 장식 왕관, 금성(♀) 하트 방패
- **색감**: 파스텔 그린 + 핑크
- **핵심 상징**: 별 12개 왕관, 밀밭, 금성 방패, 흐르는 물
- **점성**: 금성

#### 4. 황제 (The Emperor) — 위엄 있는 점하나
- **점하나 모습**: 작은 돌 왕좌에 당당하게 앉아 있는 모습. 자신감 넘치는 표정
- **배경/소품**: 양 머리 모양 귀여운 팔걸이, 앙크(☥) 지팡이, 붉은 망토, 뒤로 산
- **색감**: 파스텔 레드 + 오렌지
- **핵심 상징**: 돌 왕좌, 양 머리 장식, 앙크 홀, 험준한 산
- **점성**: 양자리

#### 5. 교황 (The Hierophant) — 스승 점하나
- **점하나 모습**: 한 손을 들어 축복 제스처. 온화한 미소
- **배경/소품**: 두 기둥, 삼중 왕관, 삼중 십자가 지팡이, 두 개의 열쇠, 앞에 작은 점하나 둘이 무릎 꿇고 있음
- **색감**: 파스텔 레드 + 골드(밝은)
- **핵심 상징**: 삼중 왕관, 삼중 십자가, 두 열쇠, 두 사제
- **점성**: 황소자리

#### 6. 연인 (The Lovers) — 사랑의 점하나
- **점하나 모습**: 점하나 둘이 나란히 서서 서로를 바라보는 모습. 눈에 하트
- **배경/소품**: 위에 날개 달린 작은 천사, 사과나무와 불꽃 나무, 밝은 정원
- **색감**: 파스텔 핑크 + 피치
- **핵심 상징**: 천사, 지식의 나무, 불의 나무, 산
- **점성**: 쌍둥이자리

#### 7. 전차 (The Chariot) — 전사 점하나
- **점하나 모습**: 작은 전차 위에 서서 정면을 응시. 결의에 찬 표정
- **배경/소품**: 별 장식 지붕, 흑백 두 마리 귀여운 스핑크스(고양이 형태), 도시 성벽
- **색감**: 파스텔 블루 + 실버
- **핵심 상징**: 흑백 스핑크스, 별 천개, 초승달 갑옷, 도시 성벽
- **점성**: 게자리

#### 8. 힘 (Strength) — 다정한 점하나
- **점하나 모습**: 귀여운 사자의 입을 부드럽게 만지며 미소짓는 모습
- **배경/소품**: 무한대(∞) 기호, 꽃 화관, 푸른 산 배경
- **색감**: 파스텔 옐로우 + 그린
- **핵심 상징**: 사자, 무한대 기호, 흰 옷, 꽃 벨트, 푸른 산
- **점성**: 사자자리

#### 9. 은둔자 (The Hermit) — 탐험가 점하나
- **점하나 모습**: 등불을 들고 산 위에 홀로 서 있는 모습. 평온한 표정
- **배경/소품**: 별(육각별) 빛나는 랜턴, 지팡이, 눈 덮인 산꼭대기, 회색 망토
- **색감**: 파스텔 그레이 + 블루
- **핵심 상징**: 등불(육각별), 지팡이, 눈 덮인 산, 회색 망토
- **점성**: 처녀자리

#### 10. 운명의 수레바퀴 (Wheel of Fortune) — 바퀴 위의 점하나
- **점하나 모습**: 큰 바퀴 위에 앉아 있는 모습. 놀란 듯 신기한 표정
- **배경/소품**: 구름 위 거대한 바퀴, 바퀴에 T-A-R-O 글자, 네 모서리에 날개 달린 작은 생물들(사자/독수리/황소/천사)
- **색감**: 파스텔 퍼플 + 골드
- **핵심 상징**: 바퀴, 스핑크스, TARO 문자, 네 생물
- **점성**: 목성

#### 11. 정의 (Justice) — 재판관 점하나
- **점하나 모습**: 정면을 바라보며 한 손에 검, 다른 손에 저울. 진지한 표정
- **배경/소품**: 두 기둥, 왕관, 붉은 망토
- **색감**: 파스텔 퍼플 + 레드
- **핵심 상징**: 양날의 검, 천칭(저울), 두 기둥, 왕관
- **점성**: 천칭자리

#### 12. 매달린 사람 (The Hanged Man) — 거꾸로 매달린 점하나
- **점하나 모습**: 나무에 한 발이 묶여 거꾸로 매달려 있으나 평온한 미소. 머리 주위 빛 후광
- **배경/소품**: 살아있는 T자형 나무(잎 달림), 빛의 후광
- **색감**: 파스텔 블루 + 그린
- **핵심 상징**: T자형 나무, 거꾸로 매달린 자세, 빛의 후광, 평온한 표정
- **점성**: 해왕성 / 물 원소

#### 13. 죽음 (Death) — 변신 점하나
- **점하나 모습**: 흰 말 위에 앉아 깃발을 들고 있는 모습. 무표정하지만 무섭지 않게
- **배경/소품**: 꽃(흰 장미)이 그려진 깃발, 지는 해, 두 탑, 새싹이 돋는 땅
- **색감**: 파스텔 라벤더 + 오프화이트 (어둡지 않게!)
- **핵심 상징**: 흰 말, 흰 장미 깃발, 두 탑 사이 석양 → 새싹/나비 등 "새 시작" 요소 강조
- **주의**: 무섭거나 어둡지 않게. "변환/새 시작"의 의미를 시각적으로 강조
- **점성**: 전갈자리

#### 14. 절제 (Temperance) — 천사 점하나
- **점하나 모습**: 작은 날개를 달고 두 잔 사이로 물을 붓는 모습. 평화로운 미소
- **배경/소품**: 한 발은 물, 한 발은 땅. 뒤로 빛나는 길이 산으로 이어짐
- **색감**: 파스텔 민트 + 골드
- **핵심 상징**: 날개, 두 개의 잔, 물과 땅, 빛나는 길
- **점성**: 사수자리

#### 15. 악마 (The Devil) — 장난꾸러기 점하나
- **점하나 모습**: 높은 곳에 앉아 장난스러운 표정. 작은 뿔 모자를 쓰고
- **배경/소품**: 역오각별(작게), 아래에 헐거운 사슬에 묶인 작은 점하나 둘 (벗을 수 있는 모습을 시각적으로 표현)
- **색감**: 파스텔 다크퍼플 + 오렌지 (밝은 톤 유지!)
- **핵심 상징**: 역오각별, 느슨한 사슬, 횃불 → 사슬이 헐겁다는 점을 귀엽게 강조
- **주의**: 무섭지 않게. 장난스러운 톤으로. "스스로 만든 속박"이라는 메시지 전달
- **점성**: 염소자리

#### 16. 탑 (The Tower) — 놀란 점하나
- **점하나 모습**: 탑에서 떨어지는데 놀란 표정 (but 귀엽게, 코믹하게)
- **배경/소품**: 작은 탑, 번개(지그재그 귀여운 형태), 왕관이 날아가는 모습, 불꽃 비
- **색감**: 파스텔 오렌지 + 레드 (밝은 톤)
- **핵심 상징**: 번개, 무너지는 왕관, 떨어지는 사람들, 바위 위의 탑
- **주의**: 재난이 아닌 "의외의 변화"로 표현. 코믹한 놀람
- **점성**: 화성

#### 17. 별 (The Star) — 소원 점하나
- **점하나 모습**: 연못가에서 두 물병으로 물을 붓는 모습. 평화로운 미소
- **배경/소품**: 밤하늘 큰 팔각별 1개 + 작은 별 7개, 연못, 나무 위 새
- **색감**: 파스텔 블루 + 옐로우(별빛)
- **핵심 상징**: 팔각별, 일곱 작은 별, 두 물병, 연못
- **점성**: 물병자리

#### 18. 달 (The Moon) — 몽환 점하나
- **점하나 모습**: 달을 올려다보는 모습. 약간 불안하면서도 신비로운 표정
- **배경/소품**: 보름달(얼굴 있는), 강아지와 늑대(둘 다 귀엽게), 물에서 올라오는 가재, 두 탑, 구불구불한 길
- **색감**: 파스텔 네이비 + 실버 (밝은 톤 유지)
- **핵심 상징**: 보름달, 개와 늑대, 가재, 두 탑, 구불구불한 길
- **점성**: 물고기자리

#### 19. 태양 (The Sun) — 신나는 점하나
- **점하나 모습**: 흰 말(유니콘/조랑말) 위에서 양팔 벌리고 환호. 최고로 밝은 표정
- **배경/소품**: 거대한 웃는 태양, 활짝 핀 해바라기들, 돌담
- **색감**: 파스텔 옐로우 + 오렌지 (가장 밝은 카드)
- **핵심 상징**: 태양, 흰 말, 해바라기, 붉은 깃발, 돌담
- **점성**: 태양

#### 20. 심판 (Judgement) — 각성 점하나
- **점하나 모습**: 구름 위에서 나팔을 부는 모습. 당당한 표정
- **배경/소품**: 십자가 깃발 달린 나팔, 아래에서 깨어나는 작은 점하나들, 눈 덮인 산
- **색감**: 파스텔 골드 + 블루
- **핵심 상징**: 나팔, 십자가 깃발, 관에서 일어나는 사람들 → 점하나들이 기지개 펴며 일어나는 모습
- **점성**: 명왕성 / 불 원소

#### 21. 세계 (The World) — 완성 점하나
- **점하나 모습**: 월계관 안에서 춤추는 모습. 양 손에 작은 지팡이. 가장 행복한 표정
- **배경/소품**: 타원형 월계관(꽃 리스), 네 모서리에 작은 동물들(사자/독수리/황소/천사)
- **색감**: 파스텔 퍼플 + 골드 + 그린 (축제 느낌)
- **핵심 상징**: 월계관, 춤추는 여인 → 춤추는 점하나, 두 지팡이, 네 생물
- **점성**: 토성
