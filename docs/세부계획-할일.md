```
홈 (테마 목록)
  ├── 오늘의 타로 (퀵 진입)
  ├── 이번 주 타로 (퀵 진입)
  ├── 카테고리별 테마 목록 (일상/연애/직장/재물/학업/기타)
  │     └── 테마 선택 → 카드 뽑기
  └── 설정 (Phase 2)

카드 뽑기
  ├── 상단: 결과 슬롯 3개 (과거/현재/미래)
  ├── 중앙: 카드 22장 그리드
  └── 하단: 다시 선택 / 결과 보기

광고 시청 (결과 게이트)
  └── 전면 광고 또는 리워드 광고

AI 해석 결과
  ├── 카드 이미지 + 카드명 + 정/역방향
  ├── AI 스트리밍 해석 (과거/현재/미래/종합 조언/한줄 요약)
  └── 공유 / 다시 보기

결과 히스토리 (로컬 저장)
  └── 이전 해석 결과 목록
```
