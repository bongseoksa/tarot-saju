## A-5. 페이지별 상태(state) 정의

- **상태**: [x] 정의 완료
- **정의 내용**: 각 화면이 가질 수 있는 모든 상태와 상태 전이

| # | 페이지 | 정의할 상태 |
|---|---|---|
| A-5-1 | 홈 | **확정**: 기본(전체 테마 표시) / 카테고리 필터 활성 / 미완료 세션 모달(+더이상 보지 않기 체크, 억제 플래그 독립 관리). 로딩/에러/빈 상태 없음(정적 데이터) |
| A-5-2 | 카드 뽑기 | **확정**: 선택 전(0/3) / 선택 중(1/3, 2/3) / 선택 완료(3/3, 하단 버튼 활성화) / 뒤집기 애니메이션 중(추가 탭 무시). 에러 없음(정적) |
| A-5-3 | 광고 게이트 | **확정**: 공통 로딩(터치 차단) → 광고 재생 → 완료 시 결과 이동 / 5초 타임아웃 실패 시 조용히 스킵 → 결과 이동 |
| A-5-4 | 결과 | **확정**: 프롬프트 요청 중(공통 로딩) → AI 스트리밍 중(타이핑 효과) → 완료(공유 활성화, 자동 저장) / 실패 시 미완료 세션 저장 → 오류 모달 → 홈. 스트리밍 중 뒤로가기 허용(미완료 세션 저장) |
| A-5-5 | 결과 히스토리 | **확정**: 결과 있음(최신순 20건, 항목 탭 → 결과 이동) / 빈 상태("여기는 아직 비어있어요. 점 하나 찍으러 가볼까요?" + 홈 이동). 로딩/에러 없음(정적). 미완료 세션 미표시 |

- **기록 위치**: docs/specs/02-taro-mvp.md (페이지별 상태 섹션 추가)
